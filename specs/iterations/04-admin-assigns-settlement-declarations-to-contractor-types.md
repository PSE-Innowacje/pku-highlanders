# Iteration 04 — Administrator can assign settlement declaration types to contractor types

## Goal
We want to have functionality, where user in role Administrator can assign settlement declaration types to contractor types

## Requirements
- We will modify "Contractor types" ("Typy kontrahentów") view
- Each contractor type should have possibility to assign multiple settlement declaration types
- Select should be done via component like MUI autocomplete with multiple values
- Reorder menu in UI - "Settlement declarations" ("Typy oświadczeń") should be on the top

## Acceptance criteria
- user in role Administrator after logging in can go the "Contractor types" view and assign settlement declarations to users
