# Iteration 02 — Administrator assigns contractor types to users

## Goal
We want to have functionality, where user in role Administrator can assign contrator types (see PRD point 4) to users in role Kontrahent.

## Requirements
- application will have menu on left hand side, where user can navigate between viewes.
- user in role Administrator will have new view "User contractor types" ("Przypisz typy kontrahentów") in webapp where he can view and edit contractor types assignments to users in role Kontrahent.
- user in role Administrator should be able to assign 1 to n contractor types to users in role Kontrahent
- assignments can be edited
- user in role Kontrahent does not have access to this view
- application should by default use Polish language

## Acceptance criteria
- user in role Administrator after logging in can go the "User contractor types" and edit assignments to users.
